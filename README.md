# Remind

## install
gem install remind

## Usage
1. Add an entry
```
  $ remind me on monday
  > the title
  > the body
  > more of the body
```
2. `remind list`

## Development
`bundle install`

## Run Tests
`rake test`

## Package gem
`rake package`
